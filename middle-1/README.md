
## Тестовое задание: Angular Todo App с дополнительной функциональностью

### Основное задание (4-8 часов)
Создать приложение для управления задачами с следующими требованиями:

#### Функциональность:
1. **CRUD операции** с задачами (создание, просмотр, редактирование, удаление)
2. **Фильтрация** задач по статусу (все, активные, завершенные, в процессе, на паузе)
3. **Поиск** по названию задачи
4. **Сортировка** по дате создания/приоритету/статусу
5. **Локальное сохранение** данных (NGRX, NGXS) например - добавить задачу в избранное
6. **GMT+3** все даты/время должны быть конвертированы в часовой пояс GMT+3
7. **Пагинация**

#### Технические требования:
- Angular 15
- TypeScript (строгая типизация)
- Reactive Forms для создания/редактирования
- Валидация полей в соответствии с сервером (+ вывод текстом ошибок валидации под полями)
- RxJS для асинхронных операций
- Любой фреймворк для UI (**Разрешено всё** кроме [Angular Material](https://material.angular.dev/))
- Responsive дизайн
- Просмотр задачи на отдельном маршруте (выводить в шаблон все данные)
- Просмотр превью задачи в модальном окне на главной странице
- Поиск по названию задачи реализуется через метод API
- Логика пагинации задач по 20 элементов на одной странице

#### Структура задачи:
```typescript
interface Task {
  id: string;
  title: string;
  description?: string;
  priority: 'low' | 'medium' | 'high';
  status: 'pending' | 'completed' | 'pause' | 'in_progress';
  createdAt: Date;
  updatedAt: Date;
  dueDate?: Date;
  image: string[];
}
```
